# ЗАКЛЮЧЕНИЕ

В рамках курсовой работы была разработана информационная система — интернет-магазин продуктов питания с доставкой. Основная задача проекта — создание удобной платформы для заказа продуктов с доставкой и простым интерфейсом для покупателей, а также инструментов для менеджеров, сборщиков и курьеров.

Разработка началась с анализа предметной области: изучены бизнес-процессы магазина до и после внедрения ИС, сформированы модели процессов (IDEF0) и диаграммы использования системы. На основе проведённого анализа определены ключевые компоненты: каталог, корзина, оформление заказа, доставка, роли пользователей и REST API для мобильных приложений сборщика и курьера.

Спроектирована нормализованная база данных (до 3НФ), содержащая 11 основных взаимосвязанных таблиц для хранения информации о пользователях, адресах, картах, категориях и товарах, заказах, корзине, сменах курьеров и логах активности. Пользовательский интерфейс веб-сайта разработан на Django Template Language с учётом удобства работы на различных устройствах.

Серверная часть реализована на Django 4.2 и Django REST Framework (Python). В качестве СУБД выбрана PostgreSQL. Для безопасности использованы встроенные механизмы Django (PBKDF2 для паролей) и аутентификация на JWT-токенах (djangorestframework-simplejwt) с ролевой моделью доступа (покупатель, менеджер, администратор, сборщик, курьер).

Приложение включает функции интернет-магазина с доставкой: каталог категорий и товаров, корзину, оформление заказа с выбором адреса и способа оплаты, расчёт стоимости доставки по координатам, бонусную систему, личный кабинет (профиль, карты, адреса), список заказов. Для персонала предусмотрены: админ-панель (Jazzmin) — управление заказами, назначение сборщика и курьера, управление товарами и категориями; аналитика по продажам (дашборд, экспорт); REST API для сборщиков (сканирование штрихкодов, комплектация, замена товара) и курьеров (приём заказа, фотоотчёт о доставке, смены).

Выполнение проекта позволило закрепить навыки разработки веб-приложений на Django, проектирования баз данных, создания REST API, работы с ролевой моделью доступа и интеграции с внешними сервисами (DaData для адресов).

Разработанная платформа обладает потенциалом для дальнейшего развития и масштабирования, способна улучшить взаимодействие с клиентами и оптимизировать процессы приёма заказов, сборки и доставки. Проект сочетает образовательную ценность и практическую применимость.
