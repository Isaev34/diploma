## Описание доступа и интерфейса

Страница каталога доступна всем пользователям, включая неавторизованных. Неавторизованным пользователям предоставляется доступ к модальным окнам (или страницам) авторизации, регистрации и восстановления пароля. Авторизованным пользователям в соответствии с их ролью предоставляется соответствующий функционал системы.

Модальные окна и разделы управления доступны только авторизованным пользователям с определёнными ролями.

**Для покупателя:**
- Просмотр каталога категорий и товаров;
- Корзина (добавление, изменение количества, удаление товаров);
- Оформление заказа (адрес доставки, способ оплаты, бонусы);
- Список заказов и просмотр заказа;
- Профиль (редактирование данных, карты оплаты, адреса доставки).

**Для менеджера:**
- Статистика по продажам (аналитика: дашборд, экспорт в CSV);
- Управление заказами в админ-панели (назначение сборщика и курьера, изменение статусов);
- Управление категориями и товарами (добавление, изменение, удаление);
- Управление остатками на складе.

**Для администратора:**
- Управление пользователями (добавление, изменение, удаление, назначение ролей);
- Управление логами активности пользователей (просмотр в админ-панели);
- Полный доступ к разделам админ-панели (каталог, заказы, корзины, адреса, карты, смены и др.).

**Для сборщика:**
- Работа через API (мобильное приложение или веб): просмотр назначенных заказов, сканирование штрихкодов, комплектация, замена товара, передача заказа в доставку.

**Для курьера:**
- Работа через API (мобильное приложение или веб): просмотр заказов на доставку, приём заказа, загрузка фотоотчёта о доставке;
- Смены (начало/окончание смены, статистика, поддержка).

Описание доступа и интерфейса может быть вынесено в Приложение А «Текст программы» или оставлено в основном тексте работы.
