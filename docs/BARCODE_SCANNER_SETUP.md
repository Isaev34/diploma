# Настройка сканера штрихкодов в Django Admin

## Описание

Добавлен функционал сканирования штрихкодов через камеру мобильного телефона в админ-панель Django для модели `Product`. Сканер работает в браузере без установки дополнительных приложений.

## Структура файлов

```
dostavkasite/
├── catalog/
│   └── admin.py (обновлен)
├── static/
│   └── catalog/
│       ├── js/
│       │   └── barcode_scanner.js
│       └── css/
│           └── barcode_scanner.css
```

## Как это работает

1. **На странице редактирования товара** рядом с полем "Штрихкод" автоматически появляется кнопка "📷 Сканировать штрихкод".

2. **При нажатии на кнопку:**
   - Открывается полноэкранное модальное окно с камерой
   - Запрашивается разрешение на доступ к камере
   - Запускается сканирование в реальном времени

3. **После успешного сканирования:**
   - Штрихкод автоматически вставляется в поле
   - Поле подсвечивается зеленым на 1 секунду
   - Сканер автоматически закрывается

## Технические детали

### Используемые технологии

- **html5-qrcode** (v2.3.8) — библиотека для сканирования штрихкодов через WebRTC API
- **WebRTC getUserMedia** — доступ к камере устройства
- **Vanilla JavaScript** — без зависимостей от jQuery или других библиотек

### Поддерживаемые форматы штрихкодов

- EAN-13 (стандарт для большинства товаров)
- EAN-8
- UPC-A / UPC-E
- CODE-128
- CODE-39
- ITF

### Поддержка устройств

✅ **Работает на:**
- Android Chrome/Firefox (задняя камера)
- iOS Safari 11+ (с ограничениями)
- Desktop Chrome/Firefox/Edge (веб-камера)

⚠️ **Требования:**
- HTTPS соединение (или localhost для разработки)
- Разрешение на доступ к камере
- Современный браузер с поддержкой WebRTC

## Использование

1. Откройте админ-панель Django
2. Перейдите в раздел "Товары" → выберите товар для редактирования
3. Найдите поле "Штрихкод"
4. Нажмите кнопку "📷 Сканировать штрихкод"
5. Разрешите доступ к камере при запросе браузера
6. Наведите камеру на штрихкод товара
7. Штрихкод автоматически распознается и вставится в поле

## Особенности для мобильных устройств

- **Полноэкранный режим** — сканер занимает весь экран
- **Адаптивный дизайн** — оптимизирован для портретной и ландшафтной ориентации
- **Задняя камера** — автоматически используется задняя камера на мобильных
- **Сенсорное управление** — удобные кнопки для мобильных экранов
- **Автозакрытие** — сканер закрывается после успешного сканирования

## Отладка

Если сканер не работает:

1. **Проверьте консоль браузера** (F12) на наличие ошибок
2. **Убедитесь, что используется HTTPS** (или localhost)
3. **Проверьте разрешения камеры** в настройках браузера
4. **Проверьте загрузку библиотеки:** в консоли должно быть `typeof Html5Qrcode !== 'undefined'`

## Безопасность

- Сканер работает только на странице редактирования товара
- Не требует серверных изменений
- Все данные обрабатываются локально в браузере
- Камера автоматически выключается при закрытии сканера
