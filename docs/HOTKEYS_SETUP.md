# Система горячих клавиш

## Что было создано

✅ **Система горячих клавиш** с поддержкой не менее 8 комбинаций
✅ **Визуальная справка** - overlay с описанием всех горячих клавиш
✅ **Контекстные действия** - разные действия на разных страницах
✅ **Уведомления** - визуальная обратная связь при использовании клавиш

## Горячие клавиши

### Навигация

| Клавиша | Действие | Описание |
|---------|----------|----------|
| `H` или `Home` | Главная страница | Переход на главную |
| `C` | Корзина | Открыть корзину |
| `P` | Профиль | Открыть профиль (требуется вход) |
| `O` | Заказы | Мои заказы (требуется вход) |
| `A` | Аналитика | Панель аналитики (только Manager/Admin) |

### Действия

| Клавиша | Действие | Описание |
|---------|----------|----------|
| `Ctrl+K` или `/` | Поиск | Фокус на поле поиска |
| `L` | Вход/Выход | Войти или выйти из системы |
| `Enter` или `Space` | Добавить в корзину | На странице товара |
| `Ctrl+Enter` | Оформить заказ | На странице оформления заказа |
| `Esc` | Закрыть | Закрыть модальные окна, убрать фокус |

### Справка

| Клавиша | Действие |
|---------|----------|
| `Ctrl+?` или `Ctrl+/` | Показать/скрыть справку по горячим клавишам |

## Итого: 11 горячих клавиш

1. `H` / `Home` - Главная
2. `C` - Корзина
3. `P` - Профиль
4. `O` - Заказы
5. `A` - Аналитика
6. `Ctrl+K` / `/` - Поиск
7. `L` - Вход/Выход
8. `Enter` / `Space` - Добавить в корзину
9. `Ctrl+Enter` - Оформить заказ
10. `Esc` - Закрыть
11. `Ctrl+?` / `Ctrl+/` - Справка

## Как использовать

### Автоматическая активация

Горячие клавиши активируются автоматически при загрузке страницы.

### Просмотр справки

1. Нажмите `Ctrl+?` или `Ctrl+/`
2. Появится overlay со всеми горячими клавишами
3. Нажмите `Esc` или кнопку закрытия для скрытия

### Приветственное сообщение

При первом посещении сайта (один раз) появится подсказка о горячих клавишах.

## Технические детали

### Файлы

- `static/js/hotkeys.js` - основной JavaScript код
- `static/css/hotkeys.css` - стили для overlay и уведомлений
- Интеграция в `templates/base/base.html`

### Особенности

- **Умная обработка**: Игнорирует нажатия, когда пользователь вводит текст
- **Контекстные действия**: Разные действия на разных страницах
- **Визуальная обратная связь**: Уведомления при выполнении действий
- **Адаптивность**: Работает на всех устройствах

### Интеграция с RBAC

- Клавиша `A` (Аналитика) работает только для Manager/Admin
- Проверка прав доступа перед выполнением действий

## Примеры использования

### Быстрая навигация

```
Пользователь на странице товара:
- Нажимает C → Переходит в корзину
- Нажимает H → Возвращается на главную
- Нажимает O → Переходит к заказам
```

### Быстрые действия

```
На странице товара:
- Нажимает Enter → Товар добавляется в корзину

На странице оформления заказа:
- Нажимает Ctrl+Enter → Заказ оформляется
```

### Поиск

```
В любой момент:
- Нажимает Ctrl+K или / → Фокус на поле поиска
```

## Настройка

### Отключение горячих клавиш

В `static/js/hotkeys.js` можно установить:

```javascript
Hotkeys.enabled = false;
```

### Добавление новых горячих клавиш

В файле `static/js/hotkeys.js` добавьте обработку в метод `handleSingleKeys` или `handleCtrlCombinations`.

## Безопасность

- ✅ Горячие клавиши не выполняют опасных действий без подтверждения
- ✅ Выход из системы требует подтверждения
- ✅ Проверка прав доступа перед выполнением действий
- ✅ Игнорирование нажатий в полях ввода

## Совместимость

- ✅ Работает во всех современных браузерах
- ✅ Поддержка Windows, macOS, Linux
- ✅ Адаптивный дизайн overlay
- ✅ Работает с Bootstrap 5

## Примечания

- Горячие клавиши не конфликтуют с системными комбинациями браузера
- При вводе текста горячие клавиши автоматически отключаются
- Esc всегда работает для закрытия модальных окон
- Приветственное сообщение показывается только один раз (использует localStorage)
